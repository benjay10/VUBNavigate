VUB Navigate

This app was developed as an asignment for “Next Generation User Interfaces”. The goal is to make a user friendly app with a new type of user interaction, in this case indirect interaction with the environment using sound and Wi-Fi signals. It uses the Google Material Design Lite framework for the lay-out and Quiet.js for ultrasonic communication. It was built with Cordova for browser and Android.

Help us make the app better! Report issues to the Github Issues page.
